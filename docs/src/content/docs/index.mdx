## Fast

Compiled Rust, no Python or Perl runtime to start per file. Formatting a
400-statement / 60&nbsp;KB Postgres file (median of 20 runs, including process
startup):

| tool | median | vs banshee |
|------|--------|------------|
| **banshee** | ~11 ms | 1× |
| sqlfluff 4.2 | ~257 ms | ~24× slower |
| pgFormatter 5.10 | ~1170 ms | ~105× slower |

Reproduce with [`scripts/bench.sh`](https://github.com/octofhir/banshee/blob/main/scripts/bench.sh).
